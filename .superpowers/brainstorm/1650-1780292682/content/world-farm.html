🌾 种田系统核心流程

① 整地
用锄头在草地 → 变成"已耕"状态 (泥土色 Tile)
② 播种
手持种子 → 点击耕地 → 消耗种子 + 体力
→ 泥土上出现小苗 (不同作物不同外观)
③ 浇水 (每日)
用水壶浇灌 → 保持湿润状态(颜色变深)
每个时段不浇水会逐渐变干
④ 生长
每天结束时 → 判断水分+节气 → 推进生长阶段
作物有 4-6 个生长阶段, 受节气影响速度
⑤ 收获
成熟后 → 右键收获 → 产出物品进背包
部分作物可连续收获 (如水稻/茄子)

🌽 作物数据结构

// CropData.tres — 作物数据 Resource
[GlobalClass]
public partial class CropData : Resource {
[Export] string CropName; // "水稻"
[Export] Texture2D[] GrowthStages; // 5 个生长阶段精灵
[Export] float BaseGrowthDays; // 基础生长天数
[Export] SolarTerm[] PreferredTerms; // 宜种节气(谷雨等)
[Export] ItemData HarvestItem; // 收获产物
[Export] bool CanRegrow; // 是否可连续收获
[Export] int RegrowDays; // 再生间隔
[Export] Season[] AllowedSeasons; // 可种植季节
}
// 耕地 Tile 状态: 普通 → 已耕 → 已耕(湿) → 已种植
🌾 节气联动示例：
水稻 → 谷雨时播种, 生长速度 +30%
小麦 → 秋分时收获, 产量 +20%
霜降后 → 所有未收作物有 50% 概率冻死 → 迫使玩家规划农时
