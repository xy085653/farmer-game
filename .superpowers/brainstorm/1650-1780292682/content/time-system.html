⏰ 二十四节气时间系统

融合中国传统文化的时间机制

时间层次结构

年 (Year) ─── 四季轮回
↓
季 (Season) ── 春 / 夏 / 秋 / 冬
↓
节气 (Solar Term) ── 每季 6 个节气，共 24 个
↓
天 (Day) ── 每个节气约 3 天游戏日
↓
时段 (Time Slot) ── 晨 / 午 / 夕 / 夜

🌿 二十四节气与四季

🌸 春

立春 → 雨水 → 惊蛰 → 春分 → 清明 → 谷雨
🌱 播种季，作物生长加速

☀️ 夏

立夏 → 小满 → 芒种 → 夏至 → 小暑 → 大暑
🌻 生长高峰，需浇水抗旱

🍂 秋

立秋 → 处暑 → 白露 → 秋分 → 寒露 → 霜降
🌾 丰收季，霜降后作物可能冻死

❄️ 冬

立冬 → 小雪 → 大雪 → 冬至 → 小寒 → 大寒
🔥 休耕季，室内制作、节日活动

🕐 一天的节奏

🌅 晨 (6:00-10:00)
☀️ 午 (10:00-16:00)
🌆 夕 (16:00-20:00)
🌙 夜 (20:00-6:00)
每个时段约 2-3 分钟现实时间。夜晚体力恢复，部分商店关门。

💻 代码设计要点

interface ITimeService {
GameTime CurrentTime { get; } // 当前时间
SolarTerm CurrentTerm { get; } // 当前节气
event Action<SolarTerm> OnSolarTermChanged; // 节气变化事件
event Action<DayPhase> OnPhaseChanged; // 时段变化事件
event Action<Season> OnSeasonChanged; // 换季事件
void AdvanceToNextDay(); // 推进到第二天
}
// 游戏时间数据对象
struct GameTime {
int Year; Season CurrentSeason;
SolarTerm Term; int DayInTerm; // 1-3
DayPhase Phase; float Minute;
}
🌾 节气对游戏的影响：
  • 每个节气有特定的"宜忌"作物——谷雨宜种水稻，白露宜收瓜果
  • 节气当天有 buff/debuff——立春耕地不耗体力，大暑浇水需求减半
  • 节气变换时触发事件（下雨、打雷、降温）
  • 全年24节气循环，每节气3天 = 一年72天
