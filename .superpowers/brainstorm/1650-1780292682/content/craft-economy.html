配方数据结构

RecipeData : Resource

WorkbenchType // 在哪制作(厨房/工坊/户外)
RequiredLevel // 需要工具等级

💰 经济系统

货币与交易

🪙 铜钱 — 基础货币 (用"文"为单位)
🏪 杂货铺 — 种子/工具/日常用品
📦 出货箱 — 放在家门口, 农产品自动出售
📜 订单板 — 村民发布需求, 额外奖励

经济循环逻辑

🌾 种田 → 🧺 收获 + 加工
→ 📦 出货箱 / 🏪 卖商店
→ 🪙 获得铜钱
→ 买新种子 🔄 买工具升级
→ 生产效率提升 → 更多产出
💰 节气影响物价: 应季作物价高

📨 事件流总览

EventBus.Publish(new DayStartedEvent()) → 所有系统推进一天
EventBus.Publish(new CropGrowthTickEvent()) → 作物生长
EventBus.Publish(new SolarTermChangedEvent("谷雨")) → 节气特效
EventBus.Publish(new ItemHarvestedEvent(item, qty)) → 收获/背包更新
EventBus.Publish(new TransactionEvent(amount)) → 金钱变动
EventBus.Publish(new StructurePlacedEvent("工坊")) → 建筑完工
💡 整体数据流：
种田产原料 → 制作系统加工增值 → 出售获利 → 购买升级 → 扩大再生产
节气系统影响每个环节：种什么、长多快、卖多少钱
